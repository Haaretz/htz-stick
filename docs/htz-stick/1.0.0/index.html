HTZ STICK

A flexible module for performant sticking of elements on scroll

Installation

jspm install htz-stick=github:haaretz/htz-stick

Usage

To initialize an instance:

import stick from 'htz-stick';

const stickyElem = stick(document.getElementById('foo'));

Parameters

Parameter Type Default Description
target  HTMLElement  -  The element to make sticky. Should be contained inside an element which will keep its size and remain static
options  Object  -  An options object
options.stickyClass  String  'is-sticky'  The class that will be attached to target when it becomes sticky
options.guardHeight  Boolean  true  Should target be wrapped by an element with an explicit height to prevent layout collapsing when target becomes position: fixed
options.stickWithin  HTMLElement  -  Limit sticky behavior to the boundaries of the stickWithin element. Default's to the stuck element's parent-parent element
options.topSpacing  Integer or HTMLElement  0  Determines how far from the top of the screen the element will stick. If topSpacing is an HTMLElement, the element's height will be calculated and used
options.stickAfter  HTMLElement  -  Only make elem sticky after stickAfter reaches sticking position
options.delta  Integer  0  Only stick if scroll delta exceeds provided amount.
options.direction  String  'both'  Only stick when scrolling in indicated direction. Can be one of 'both', 'up' or 'down'
options.secondaryAction  Function  -  A custom callback to run whenever evaluating the scroll event

Events

See documentation for a detailed list of custom events fired.

Instance Methods

The following methods are available in every instance returned by htz-stick:

Method Description
target  The target element
isSticky  Returns true if target is currently sticky, false otherwise
destroy  Remove all classes, style and event listeners associated with an instance
init  Reinitialize an instance
