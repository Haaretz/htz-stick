(require("htz-stick"))(target, options) → {module:htz-stick#API}

Make an element stickable

Parameters
Name Type Description
target HTMLElement

The element to make sticky. Should be contained inside an element which will keep its size and remain static.
options Object
Properties
Name Type Attributes Default Description
stickyClass String <optional>
 'is-sticky'

The class that will be attached to target when it becomes sticky.
guardHeight Boolean <optional>
 true

Should target be wrapped by an element with an explicit height to prevent layout collapsing when target becomes position: fixed.
stickWithin HTMLElement <optional>

Limit sticky behavior to the boundaries of the stickWithin element. Default's to the stuck element's parent-parent element.
topSpacing Integer | HTMLElement <optional>
 0

Determines how far from the top of the screen the element will stick. If topSpacing is an HTMLElement, the element's height will be calculated and used.
stickAfter HTMLElement <optional>

Only make elem sticky after stickAfter reaches sticking position.
delta Integer <optional>
 0

Only stick if scroll delta exceeds provided amount.
direction String <optional>
 'both'

Only stick when scrolling in indicated direction. Can be one of 'both', 'up' or 'down'.
secondaryAction module:htz-stick#secondaryAction <optional>

A custom callback to run whenever evaluating the scroll event.
