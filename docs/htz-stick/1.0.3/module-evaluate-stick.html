A utility for evaluating an element's stickiness

evaluate-stick.js:31

(require("evaluate-stick"))(target, stickWithin, topSpacing, lastWinScrollY, winScrollY, offsetElem, delta, direction, isSticky) → {Object}

Evaluate if an element should be sticky, and the values to assign to the css position and top properties.

Parameters
Name Type Description
target HTMLElement

The element to evaluate
stickWithin HTMLElement

The container that sets stick limits
topSpacing HTMLElement | Number

The distance from the top of the viewport target should stick in, or an element to stick to.
lastWinScrollY Number

The last known scroll position before the current one.
winScrollY Number

The current scroll position.
offsetElem HTMLElement

An element to determine sticking position's top limit.
delta Integer

Only stick if scroll delta exceeds provided amount.
direction String

Only stick when scrolling in indicated direction. Can be one of 'both', 'up' or 'down'.
isSticky Boolean

Indicates if target is currently sticky.
Returns

Evaluated results. Has makeSticky, newPosition and newTop properties.

Type
Object
